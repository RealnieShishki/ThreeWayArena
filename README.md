# ?? ThreeWay Arena

2D Roguelite PvP (1v1v1) built with Godot 4.2

## ?? Structure
- /assets - Raw assets (sprites, sounds)
- /scenes - Godot scenes (.tscn)
- /scripts - GDScript files (.gd)
- /exports - Build outputs

## ?? Setup
1. Open Godot 4.2
2. Import project from this folder
3. Run 'test_room.tscn'
